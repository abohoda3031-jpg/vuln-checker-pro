# Vulnerability Checker

## Features
- Scans software for known vulnerabilities.
- Supports a wide range of vulnerability databases.
- Generates detailed reports for identified issues.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/abohoda3031-jpg/vuln-checker-pro.git
   ```
2. Navigate into the directory:
   ```bash
   cd vuln-checker-pro
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
After installation, you can run the vulnerability checker with:
```bash
npm run start
```

You can specify options such as:
- `--path` to define the path to scan.
- `--verbose` for detailed output.

## Supported Vulnerabilities
- SQL Injection
- Cross-Site Scripting (XSS)
- Command Injection
- Buffer Overflow
- Remote Code Execution
